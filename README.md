# Dotfiles

My dotfiles:
1. Oh My Zsh
1. Software installs
1. macOS preferences

## Oh My Zsh

```
sh oh-my-zsh.sh
```

## Software installs

```
sh install.sh
```

## macOS preferences

```
./.macos
```

> Make file executable > chmod u+x .macos
